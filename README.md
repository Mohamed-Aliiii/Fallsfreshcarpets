## Optional images to add later
Place these in the images folder if you want real photo backgrounds:
- hero-living-room-carpet.jpg
- commercial-cleaning.jpg

The website works without them because CSS fallback backgrounds are included.

## GitHub Pages
1. Upload all files to a GitHub repository.
2. Go to Settings > Pages.
3. Source: Deploy from branch.
4. Branch: main.
5. Folder: / root.
6. Save.
